# PagePen - Web Annotator & Screenshot

A lightweight Chrome extension to draw, highlight, annotate and screenshot any webpage.

## Features

- Freehand pen drawing
- Highlighter
- Shapes (rectangle, circle, arrow, line)
- Text annotations
- Color picker and stroke width
- Screenshot capture of the annotated page
- Toolbar position: floating, pinned left, pinned right

## Privacy

PagePen runs entirely in your browser. It does not collect, store or transmit any personal data. See [privacy-policy.html](privacy-policy.html) for details.

## Permissions

- `activeTab` — overlay the drawing canvas on the current page
- `scripting` — inject the drawing tools on demand
- `downloads` — save your annotated screenshots as PNG files
